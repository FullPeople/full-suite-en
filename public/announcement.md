## Modules currently bundled [todo] [en]

- <span style="color:#c89251">**Bestiary**</span>: drag a monster onto the map to spawn it; auto-joins initiative; DM gets a stat block popover when the token is selected
- <span style="color:#c89251">**Character Cards**</span>: click-to-roll dice, in-place HP/AC editing, JSON import/export
- <span style="color:#5dade2">**Global Search**</span>: self-hosted SRD / homebrew libraries (configure your own data source URL)
- <span style="color:#5dade2">**Initiative Tracker**</span>
- <span style="color:#5dade2">**Dice Tools**</span>: history log, hit/crit particle effects, replay
- <span style="color:#c89251">**HP / AC Bubbles**</span>: drop-in replacement for the third-party Stat Bubbles plugin, with silhouette mode for hidden enemies
- <span style="color:#c89251">**Portals**</span>: click-to-teleport tokens
- <span style="color:#5dade2">**Time Stop**</span>: visual effect, DM announcement, scene-wide audio mute
- <span style="color:#5dade2">**Status Tracker**</span>: drag buff pills from the palette onto tokens to apply / toggle
- <span style="color:#5dade2">**HP Bar component**</span>: lightweight standalone HP/AC popover, part of the HP / AC Bubbles feature
